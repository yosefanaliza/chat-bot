# Simple Page with Gallery

A clean and responsive web page featuring a photo gallery.

## Features

- Simple, centered welcome section
- Responsive image gallery with 6 photos
- Hover effects on gallery images
- Mobile-friendly design
- Images powered by Picsum Photos

## Files

- `index.html` - Main HTML structure
- `style.css` - Styling and layout

## Usage

Simply open `index.html` in your web browser to view the page.

## Customization

To use your own images, replace the Picsum URLs in `index.html` with your own image paths:

```html
<img src="path/to/your/image.jpg" alt="Your description">
```

## Technologies

- HTML5
- CSS3
- Grid Layout
- Picsum Photos API
